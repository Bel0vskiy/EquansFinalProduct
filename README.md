# EQUANS Final Product - MEP Generator

Integrated solution for automated MEP element placement using GNN and KNN models.

## Quick Start

1. **Install Dependencies**: `pip install -r requirements.txt`
2. **Run UI**: `python main/UI/main_UI.py`
